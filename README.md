# WebGIS Fasilitas Pendidikan Kota Malang

## Deskripsi
Proyek WebGIS fasilitas pendidikan Kota Malang untuk Ujian Tengah Semester mata kuliah Kapita Selekta Sistem Informasi.

WebGIS ini digunakan untuk menampilkan persebaran fasilitas pendidikan di Kota Malang secara interaktif menggunakan peta digital berbasis OpenStreetMap.

---

## Wilayah & Tema
- Wilayah: Kota Malang, Jawa Timur
- Tema: Persebaran Fasilitas Pendidikan

---

## Fitur
- Peta interaktif fasilitas pendidikan
- Popup informasi sekolah
- Filter berdasarkan kategori dan kecamatan
- Dashboard statistik pendidikan
- Visualisasi grafik distribusi fasilitas pendidikan
- Layer control dan legenda peta
- Routing lokasi sekolah

---

## Teknologi
Project ini dibangun menggunakan:
- HTML
- CSS
- JavaScript
- Leaflet.js
- OpenStreetMap
- Chart.js

---

## Data
Data fasilitas pendidikan bersumber dari:
- OpenStreetMap (OSM)
- Overpass API / HOT Export
