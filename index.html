Tentang EduMap Malang

Solusi Digital untuk Pencarian Fasilitas Pendidikan

Tujuan

EduMap Malang bertujuan membantu masyarakat Kota Malang dalam mencari dan membandingkan fasilitas pendidikan terdekat melalui peta interaktif yang mudah digunakan.

Sumber Data

Data yang digunakan bersumber dari OpenStreetMap (OSM) yang diekspor melalui Raw Data API oleh Humanitarian OpenStreetMap Team (HOT).

Teknologi

Dibangun menggunakan HTML, CSS, JavaScript, Leaflet.js untuk peta interaktif, Chart.js untuk visualisasi data, dan OpenStreetMap sebagai basemap.

Lisensi Data

Data tersedia di bawah Open Database License (ODbL). Hak individu dalam database dilisensikan di bawah Database Contents License.
